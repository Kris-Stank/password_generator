# Генератор безопасных паролей

Проект представляет собой программу, которая генерирует безопасные пароли с учетом пользовательских предпочтений. Программа позволяет задавать количество паролей, длину пароля и настраивать, какие символы должны быть включены или исключены.

## Описание

Программа позволяет пользователю:
- Выбрать количество паролей для генерации.
- Определить длину каждого пароля.
- Настроить включение цифр, прописных и строчных букв, а также специальных символов.
- Исключить неоднозначные символы (например, "il1Lo0O").

Генератор создает пароли с использованием выбранных пользователем символов, обеспечивая безопасность и уникальность каждого пароля.

## Как запустить

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Kris-Stank/password_generator
```

2. Перейдите в директорию с проектом:
   ```bash
   cd password_generator
```

3. Запустите программу:
```bash
python password_generator.py
```
